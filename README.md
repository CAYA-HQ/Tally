# Tally

Tally is a modern business management application built to help small business owners, farmers, students, and professionals organize and manage their daily operations more effectively.

Many small businesses struggle with keeping proper records, tracking inventory, remembering important tasks, and managing stock levels. Tally was created to simplify these processes through an intuitive and easy-to-use platform.

The application combines inventory management, reminders, notifications, and business insights into one centralized system, making it easier for users to stay productive and make informed decisions.

---

# Features

## User Authentication

Tally provides a secure authentication system that allows users to create accounts, log in safely, and access protected features. Authentication is handled using JWT and encrypted passwords.

### Includes:
- User registration
- Secure login
- JWT authentication
- Protected routes
- Password encryption

---

# Inventory Management

The inventory system allows users to keep track of products, quantities, prices, and stock levels in real time.

Users can:
- Add inventory items
- Edit and update products
- Delete products
- Track quantities
- Set low stock thresholds
- Monitor inventory value
- Search and filter items

This helps reduce losses caused by poor stock management and makes it easier to monitor business performance.

---

# Smart Reminders

Tally includes a reminder system that helps users stay organized and avoid forgetting important tasks.

Users can:
- Create reminders
- Set due dates
- Assign priorities
- Mark reminders as complete
- Filter pending and completed reminders

---

# Notification System

The application provides notifications for important business activities and updates.

Examples include:
- Low stock alerts
- Reminder due notifications
- Task updates
- Inventory warnings

---

# Dashboard & Insights

Tally provides a dashboard that gives users a quick overview of important business information.

The dashboard includes:
- Total inventory items
- Inventory value
- Low stock statistics
- Pending reminders
- Recent activities

The goal is to help users make better decisions using simple business insights.
